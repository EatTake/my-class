# Checklist

## 信息窗宽度约束
- [x] 信息窗宽度在所有屏幕尺寸下均不超过黑板区域宽度
- [x] 响应式布局下无横向滚动条出现
- [x] 移动端竖屏模式下信息窗宽度正确约束

## 学生属性弹窗溢出显示
- [x] 弹窗可以超出 `.desks-area` 边界显示
- [x] 弹窗内容完整显示，无被裁剪
- [x] 点击弹窗外部区域时弹窗自动隐藏
- [x] 弹窗 `z-index` 足够高，不被其他元素遮挡

## 移动端触发方案
- [x] PC端鼠标悬停触发弹窗正常工作
- [x] 移动端点击触发弹窗正常工作
- [x] 移动端再次点击或点击其他区域弹窗隐藏
- [x] PC端和移动端交互逻辑正确区分
- [x] 主流移动设备浏览器兼容性测试通过
